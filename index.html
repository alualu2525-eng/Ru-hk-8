2026 港澳極致導航

絕不精簡一字一句 / 小馬哥家庭團

1. 飯店 ➔ 竺扶大班燒味

🚇 葵興站搭 荃灣線(中環方向) ➔ 「金鐘站」 轉 港島線(柴灣方向) ➔ 「灣仔站」A3出口。
👣 出站後過馬路，沿 灣仔道 步行約 5 分鐘抵達 竺扶大班燒味。

2. 竺扶大班 ➔ 半山扶梯

👣 離開餐廳 右轉 沿灣仔道走 ➔ 看到 菲林明道左轉 ➔ 走到底在大街(莊士敦道)等車。
🚃 找到「菲林明道 42E」叮叮車站 ➔ 搭乘 西行車(往堅尼地城/屈地街) ➔ 在 「72W 租庇利街」 下車。
👣 下車過馬路進入 租庇利街 直行 ➔ 穿過德輔道中 ➔ 抵達 半山扶梯。

3. 太平山下山 ➔ 天星小輪 ➔ 佳佳甜品 ➔ 樂酒場

🚌 山頂總站搭 15 號巴士 下山 ➔ 在 「中環 5 號碼頭」 下車 ➔ 步行至隔壁 7 號碼頭。
⛴️ 搭 天星小輪 往尖沙咀 ➔ 抵港後沿梳士巴利道走 ➔ 進 尖沙咀站 搭地鐵至 「佐敦站」A出口。
👣 A出口出來 右轉 沿佐敦道走 ➔ 第一個路口 右轉 白加士街 ➔ 直走 3 分鐘抵達 佳佳甜品 (寧波街口)。
🍻 往廟街方向走 ➔ 抵達 樂酒場 居酒屋。

1. 寄存行李 ➔ 一點心

🕒 09:00 旭逸退房 ➔ 搭接駁車至葵興站 ➔ 地鐵至 「太子站」D出口。
👣 出站直走 長沙灣道 ➔ 到上海街路口 左轉 ➔ 步行至 上海街 695 號門市 寄存行李。
👣 離開寄存點 左轉 上海街 ➔ 走到運動場道路口 右轉 ➔ 抵達 一點心。

2. 採買行程：珍妮 ➔ 兆成行 ➔ 蘭芳園

🚇 太子站搭地鐵至 「上環站」E2出口 ➔ 出站過馬路入 永和街 ➔ 抵達 珍妮曲奇。
👣 離開珍妮 右轉 出巷口 ➔ 右轉 蘇杭街直走 2 分鐘 ➔ 抵達 兆成行。
👣 離開兆成行，沿蘇杭街往中環方向走 ➔ 接大路德輔道中 ➔ 右轉 進入 租庇利街 ➔ 沿半山扶梯旁的小路(結志街)走 ➔ 抵達 蘭芳園。

3. 前往澳門：港澳快線

🚇 中環/上環搭地鐵回 太子站 ➔ 回上海街 695 號 領取行李 ➔ 現場搭乘 港澳快線 跨境巴士 ➔ 抵達澳門口岸/飯店。

4. 澳門入住後：禮記 ➔ 大三巴 ➔ 最香餅家

🏨 利澳酒店 入住放行李。
👣 步行 10 分鐘往 荷蘭園大馬路 ➔ 趁 20:00 前吃 禮記雪糕。
👣 往西南方(噴水池方向)走 ➔ 抵達 大三巴牌坊。
👣 離開大三巴，往南方走進入 夜呣街 12 號 ➔ 買 最香餅家 (這家很早關，一定要先衝)。

5. 晚餐 ➔ 夜景 ➔ 宵夜 ➔ 回飯店

📍 晚餐：陳光記燒鵝 ➔ 約翰四世大馬路搭 MT1/MT2 至 新濠影滙(8字摩天輪)。
🚠 步行至 永利皇宮 看水舞 ➔ 至 路氹西站 搭輕軌遊夜景。
🚌 搭車回新馬路 ➔ 吃 杏香園 ➔ 步行回 利澳酒店。

1. 世紀咖啡 ➔ 官也街

🎒 10:00 退房 ➔ 行李寄存利澳酒店前台。
☕ 搭 3 號巴士至「新馬路」 ➔ 沿 東方斜巷 爬坡至 世紀咖啡(崗頂店)。
🚌 搭車往氹仔 ➔ 在 官也街 站下車 ➔ 散策、伴手禮。

2. 金光大道：威尼斯人 ➔ 倫敦人

👣 官也街走 望德聖母灣自動步行系統 ➔ 直達 威尼斯人。
👣 穿過室內天橋或步行 ➔ 抵達 倫敦人 拍大笨鐘、夜景。

3. 領行李 ➔ 機場

🚕 從倫敦人搭計程車(或巴士) 返回利澳酒店領取行李 (這點極重要！)。
🚌 領完行李，在酒店門口搭接駁車或巴士 ➔ 抵達 澳門機場 辦理登機。

💳 證件財物

護照、台胞證、港簽(2份)、機票電子檔、港幣現金(5000+)、實體八達通、利澳/旭逸憑證。

🔌 電子設備

英規轉接頭x3、快充頭、Apple Watch/iPhone線、2萬內行動電源、澳門SIM卡。

💊 應急藥品

EVE止痛、感冒藥、止瀉藥、太田胃散、小林退熱貼、酒精棉片、OK繃。

● 洗杯水：茶餐廳第一桶茶水是洗筷子的，別喝。

● 電子菸：絕對不可入香港境，罰款 5 萬元起。

● 澳門交通：計程車易坑人，多利用發財車與輕軌。

● 下車按鈴：公車要提早按鈴，司機才停。

● 手扶梯：香港靠右站，左側讓趕路人。
