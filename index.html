— 完整行李清單 —

  • 護照 / 港簽 / 台胞證 (正本)
  • 港幣 / 澳門幣 現金 (建議各備 2000+)
  • 八達通實體卡 (必備)
  • 英規轉接頭 (三腳扁型) x 2
  • 行動電源 (20000mAh) + 各式充電線
  • 漫遊卡 / Wifi 分享器
  • 隨身面紙 / 酒精濕巾 (餐廳多不提供)
  • 薄外套 (飯店/商場冷氣極強)
  • 常用藥 (腸胃/止痛/過敏)

— 大師級重要叮嚀 —

⭐ 叮叮車： 後門上車不用刷卡，前門下車才刷八達通。

⭐ 洗筷子文化： 香港餐廳給的第一壺熱水/茶通常是洗餐具用的，別喝。

⭐ 最香餅家： 這是本次行程最難買的東西，只要看到有人排隊請立刻跟進，不要猶豫。

⭐ 澳門交通： 善用「發財車」(賭場接駁車)，若要去利澳，搭到外港碼頭或回力、金沙是最快的。

⭐ 併桌： 香港餐廳併桌是常態，包包不要放旁邊座位。
