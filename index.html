# 交通心法

香港地鐵出口非常複雜，務必「眼看標示、腳不動」。叮叮車是「後門上車、前門刷卡下車」，不要走錯。

# 美食避雷

許多老店不收 1000 元港幣大鈔，請多準備 100 或 500 元。佳佳甜品排隊很快，看到人多不要怕，先去拿號碼牌。

# 澳門生存

澳門發財車有些需要賭場會員卡（如威尼斯人），建議先備好護照辦理免費會員卡。杏香園隱身在福隆新街，那是拍電影的好地方。
